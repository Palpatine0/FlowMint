# FlowMint

A personal finance tracker built with React and TypeScript. Track income, expenses, and visualize your spending habits — all from your browser with no server required.

## Features



## Tech Stack

- **React 19** + **TypeScript**
- **Vite** for build tooling
- **Tailwind CSS v4** for styling
- **Recharts** for data visualization
- **Lucide React** for icons

## Getting Started

```bash
cd web
pnpm install
pnpm run dev
```
